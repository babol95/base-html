Base HTML

Pourquoi ?

Parce que les technologies evolues et qu'il faut arréter de se prendre la tête pour rien.

Comment l'utiliser ?

Base HTML est diponible sur github. Il suffit donc de clôner le depot puis d'executer 3 commandes dans le terminal

npm install
bower install
gulp


Base HTML est désormais pret à être utilisé

Utilitaires disponibles

Bower

Bower permet de télécharger directement dans le projet des librairies CSS/JS. Cela facilitie l'installation, fait gagner du temps et garde les librairies à jour.
Site officiel Bower

Gulp (avec Elixir)

Gulp permet d'automatiser certaines taches en rapport au css/js comme la concaténatation de fichier ou encore la compilation de fichier sass.
Site officiel Gulp Site officiel Elixir

Structure de dossier

  • bower_components | Librairies téléchargé via BOWER
  • node_modules | Librairies Node
  • pubic | HTML
  • resources | Sass, Css, Js etc ..
  • .gitignore | Github
  • bower.json | Dépendences bower
  • gulpfile.js | Instructions Gulp
  • packages.js | Dépendences Node
  • README.md | Github
Les dossiers qui nous interessent sont :
bower_components

Ce dossier contient toutes les librairies téléchargés via bower.

Public

Ce dossier contient tout le html des pages web ainsi que les fichiers css et js concaténé par "gulp".

Resources

Ce dossier contient toutes les resources pour la création du site a savoir les librairies css et js mais aussi le code personel.

Sass

Sass simplifie la création de feuilles de style grâce à une syntaxe allégé et la possibilité d'utiliser des fonctions et variables

Comment l'utiliser

Tout les fichiers sass sont dans le dossier "resources/assets/sass"
Le dossier comprend un fichier app.sass qui permet d'importer les autres fichier sass
Pour compiler les fichier sass il faut utiliser la commande gulp ou bien gulp watch. La seconde commande permet de "surveiller" les fichiers sass et à chaque modification le code sera recompilé.
Pour plus d'information à propos de l'utilisation de SASS Voir le site officiel

Bower

Bower permet de télécharger directement dans le projet des librairies css/js

Comment l'utiliser

Pour télécharger une librairie il faut tout d'abord la chercher grâce à la commande bower search suivie du nom de la librairie ex bower search jquery
Une fois la librairie trouvé il faut la téléchargé grâce à la commande bower install suivie du nom de la librairies ex bower install jquery
La librairie est maintenant téléchargé dans le dossier "bower_components" il faut donc la copier grâce à gulp.

Gulp

Gulp permet d'automatiser certaines tâche en rapport au css/js/sass

Comment l'utiliser

Dans le fichier gulpfile.js sont ecris les tâches à exectuer tel que :

  • Copier une librairie
  • Compiler le sass
  • Concaténé le CSS
  • Concaténé le JS

Des exemples sont inclus dans le fichier gulpfile.js
